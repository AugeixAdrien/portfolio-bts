Deezer Rich Presence

Rich presence est une fonctionnalité de discord permettant d'afficher des informations sur le jeu auquel l'utilisateur jour ou sur la musique que vous écoutez, cependant, le seul service proposant l'affichage de la musique en cours nativement est Spotify. J'ai donc décidé de créer une application permettant d'afficher les informations de la musique que vous écoutez sur Deezer sur votre profil discord.

Cette application m'aura demandé plusieurs essais avant d'obtenir la version actuelle, voici les 3 versions :



1. Utilisation d'un extension de navigateur

La première version de mon programme utilisait une extension de navigateur que j'ai modifée pour qu'elle corresponde à mes besoins et envoie les données des musique à mon application python via des requetes HTTP. Cependant, cette méthode n'était pas optimale et assez bancale. De plus, je préfère utiliser l'application de bureau Deezer (bien que celle-ci ne soit qu'un portage de l'application web) et non le site web Deezer, ce qui rendait cette méthode inutilisable.



2. Utilisation de l'API de deezer

Afin d'utiliser l'application de bureau Deezer, j'ai donc décidé de passer par l'API de Deezer avec le module requests de python. Cependant, l'API de deezer ne permet pas de récuperer les informations de la musique en cours, mais uniquement des informations diverses, telles que la dernier musique écoutée, la dernière musique ajoutée en favori, etc.. Pour l'utilisation que j'en faisais cela m'est convenu quelques mois, mais ne pas être parvenu à mon but m'a poussé à chercher une autre solution.



3. Utilisation de l'API Windows Runtime et de l'API de deezer

Windows Runtime est une API permettant de récupérer des informations sur windows. Dans le cadre de mon projet, le nom du media actuellement lu ainsi que l'application dont le media provient. Pour cela, j'ai utilisé le module winrt. Je vérifie donc si le media actuellement lu provient de Deezer, si c'est le cas, je récupère le titre et l'artiste de la musique, je fais en suite un appel vers l'API de deezer pour récupérer l'image de l'album ainsi que le lien de la musique pour l'afficher sur discord.



Conclusion


J'ai donc réussi à créer une application permettant d'afficher les informations de la musique actuellement lue sur Deezer sur discord.Ce projet, en plus d'être parvenu à mon but, m'a permis de découvrir de nouvelles choses, notamment l'utilisation de l'API Windows Runtime.Celà m'a aussi poussé à chercher des alternatives à une simple application. Je suis assez satisfait du résultat final, même si je pense qu'il est possible d'améliorer le code et l'application en général.
